# [Xiang An's Personal Homepage](https://anxiangsir.github.io/)

## 本地部署

1. **克隆仓库**

   ```bash
   git clone https://github.com/anxiangsir/anxiangsir.github.io.git
   cd anxiangsir.github.io
   ```

2. **启动本地服务器**

   使用 Python 内置的 HTTP 服务器：

   ```bash
   # Python 3
   python -m http.server 8000
   ```

   或者使用 Node.js 的 live-server（需要先安装）：

   ```bash
   npm install -g live-server
   live-server
   ```

3. **访问网站**

   在浏览器中打开 http://localhost:8000 即可查看网站。
